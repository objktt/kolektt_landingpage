## 이미지 요구사항

- **해상도**: 최소 500x500px (정사각형 권장)
- **포맷**: JPG 또는 PNG
- **품질**: 고해상도 (바이닐 레코드 크기로 표시되므로)
- **저작권**: 공정 이용 가능한 이미지 사용

## 다운로드 소스 추천

1. **Discogs** - 고품질 앨범 커버 이미지
2. **AllMusic** - 공식 앨범 아트워크
3. **위키피디아** - 공정 이용 이미지
4. **Last.fm** - 앨범 커버 데이터베이스
5. **MusicBrainz Cover Art Archive** - 오픈 소스 커버 아트

## 설치 방법

```bash
# albums 폴더 생성 (이미 생성됨)
mkdir -p public/albums

# 각 이미지를 해당 폴더에 저장
# 예: public/albums/abbey-road.jpg
```

모든 이미지가 준비되면 웹사이트에서 30개의 아이코닉한 앨범 커버들이 3D 애니메이션과 함께 표시됩니다!
